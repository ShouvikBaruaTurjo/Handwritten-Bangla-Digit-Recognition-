# Handwritten-Bangla-Digit-Recognition-
A project that enables users to draw Bangla digits on an interactive interface, upon which a trained machine learning model predicts the corresponding digit in real-time.
The project involves training a machine learning model to recognize Bangla digits drawn by users. Google Colab was utilized for development, with a T4 GPU to accelerate training. The numeric subset of the Ekush dataset was used for model training and evaluation.
